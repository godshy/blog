## Process 
### To Create a post
Under source/_posts
``` bash
$ npx hexo new post "NAME"
```
Under source/_draft
``` bash
$ npx hexo new draft "NAME"
```
Under source
``` bash
$ npx hexo new page "NAME"
```

Test it
``` bash
$ npx hexo server
```

Generate it 
``` bash
$ npx hexo generate
```

Install deploy manager
``` bash
npm install hexo-deployer-git --save
```

Deploy it 
``` bash
$ npx hexo deploy
```
